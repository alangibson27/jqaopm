Why? (31-Oct-2016)

There are plenty of perfectly adequate Spectrum emulators already freely available, so why did I feel the need to write +F? It did all start with the idea to make possible 2-player Spectrum gaming across a network - which, as far as I was aware at the time I started out, wasn’t possible in any other existing Spectrum emulator - but that wasn’t my only motivation. Mainly, I wanted to learn, and to have a project of my own to sink my teeth into, experiment with, and give me a place to have a go at applying things that I have read or thought about.
