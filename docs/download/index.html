Available packages

  • 64-bit Windows Installer
    • This is the simplest way to get up and running on a compatible Windows system.
    • On Windows 10, you might see a warning from Windows SmartScreen when you run the installer. Just click the “More Info” text and then the “Run anyway” button.
  • Universal Zip Package
    • This download is suitable for all operating systems. You’ll need to install a recent version of Java 8 separately, in order to run. You can download this for free from www.java.com
  • Debian Package
    • Prebuilt package for Ubuntu, Raspbian and other Debian-based Linux distributions.

All of these packages contain images of the 48k and +2 Spectrum ROMs, which are required in order for +F to run.
